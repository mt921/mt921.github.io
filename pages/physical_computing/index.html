CHIP - The Card Dealing Robot

This is a group project in collaboration with Rhea Park which involves developing our skills in CAD, electronics, rapid manufacturing techniques and physical prototyping. The concept is CHIP, a card dealing robot which can help support the gameplay for Poker, BlackJack & UNO by rotating to face the required players and then either dealing a card face up or face down.

CAD modelling

The initial CAD model for the base components was created in Fusion360. This setup includes all of the mechanical parts required for rotation of the body.

Physical Prototyping

Our initial physical prototype included all of the CAD parts with the addition of an arduino Mega a motor driver chip & several supports. The prototype successfully showed that CHIP could turn to face any user that wanted a card:




Next Steps

This project is currently a work in progress, our next steps are detailed below:
  1. Develop refined electronics schematics to streamline prototyping process
  2. Redesign supporting structures to make pin access easier
  3. Develop the mechanical structure for a card flipping mechanism
  4. Implementing an interactive OLED display
  5. Finalise assembly with supporting walls & design features
