Physical Computing & Robotics

  • Concept
  • CHIP is a card dealing robot which can help to support tabletop gameplay by rotating to face the desired player. This project was built in collaboration with Rhea Park and involved developing our skills in CAD, electronics, prototyping techniques (i.e. 3D printing, laser cutting), working with C++ and the Arduino platform.
  • Rotating Base
  • The initial CAD model for the base components was created in Fusion360. This setup includes all of the mechanical parts required for rotation of the body.
  • A test with an initial prototype shows the robot can turn to face the required player.
