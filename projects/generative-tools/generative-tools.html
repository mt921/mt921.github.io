Generative Colour Tools

  • The overall aim of this project is to generate cohesive colour schemes that fit into different categories e.g. harmonious, analogous, triadic etc.
  • Several concepts were explored including the use of procedural generation, noise & random distributions, colour spaces and colour classification i.e. with the use of TensorFlow's neural networks.
  • 1. Random Generation
  • Initial investigations explored the concept of random generation. This pattern generates varying strengths of blue in a grid structure.
